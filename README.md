# DDTP Translate

Debian Description Translation Project tool with DDTSS-style review UI.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install ddtp-translate
```
